Chip type: MoGene-1_1-st-v1

Affymetrix Mouse Gene 1.1 ST Array Plate

"The Affymetrix Mouse Gene 1.1 ST Array Plate enables high-throughput expression profiling of 16, 24, or 96 samples at a time using the same content as the industry-standard GeneChip Mouse Gene 1.0 ST Array."

Note that Affymetrix does not provide a CDF file for this chip type. Instead, the following CDF has been created from a corresponding Bioconductor annotation package.

> cdf <- AffymetrixCdfFile$byChipType("MoGene-1_1-st-v1", tags="pd.mogene.1.1.st.v1,HB20101204")
> cdf
