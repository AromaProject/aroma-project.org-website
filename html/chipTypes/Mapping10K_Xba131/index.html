Chip type: Mapping10K_Xba131

Note the difference between Mapping10K_Xba131 (aka 'Mapping 10K Array') and Mapping10K_Xba142 (aka 'Mapping 10K Array 2.0').

The Mapping10K_Xba131 chip interrogates 11,564 SNPs.
