Chip type: Axiom_Exome319

The Axiom Genotyping Solution

More at http://www.axiom.affymetrix.com/.

> cdf <- AffymetrixCdfFile$byChipType("Axiom_Exome319", tags="r1")
> cdf

AffymetrixCdfFile:

> unitNames <- getUnitNames(cdf)
> seqToHumanReadable(grep("^AFFX-", unitNames))
[1] "1-85116"
> seqToHumanReadable(grep("^AX-", unitNames))
[1] "85117-404099"

NOTE: There are no analysis methods in aroma.affymetrix that supports this chip type. There are several reasons for this, one is that the CEL files the Axiom platform are so called multi-channel CEL, and none of the preprocessing methods are prepared for this.
