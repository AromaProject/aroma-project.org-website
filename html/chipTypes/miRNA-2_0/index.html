Chip type: miRNA-2_0

> cdf <- AffymetrixCdfFile$byChipType("miRNA-2_0")
> cdf
AffymetrixCdfFile:
Path: annotationData/chipTypes/miRNA-2_0
Filename: miRNA-2_0.CDF
Filesize: 6.08MB

RAM: 0.00MB
File format: v4 (binary; XDA)

Note that the default CDF provided by Affymetrix is in the text/ASCII file format. It is highly recommended to convert this to the binary/XDA format using convertCdf() of the affxparser package.
