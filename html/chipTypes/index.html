Axiom Genotyping Solution chip types

Note: None of the Axiom chip types are supported by the Aroma framework. The reason for this is that the Affymetrix Axiom technology is quite different from the classical chip types. For instance, Axiom is a multi-channel technology, whereas all other chip types are using a single-channel technology.
