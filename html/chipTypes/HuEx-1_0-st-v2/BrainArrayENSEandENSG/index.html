Downloads:

  • HuEx-1_0-st-v2,U-ENSG,G-ENSE,v14.1,SP20120129.cdf.gz - Custom CDF. Contributed by Steve Pederson (School of Mathematical Sciences, The University of Adelaide), January 2012.
  • HuEx-1_0-st-v2,U-ENSG,G-ENSE,v.14.1,SP20120129.supportfiles - Contains the probe_tab file & mapping of probes to genomic co-ordinates for genome build GRCh37/hg19.
  • The script used to build the above is available at: buildCustomCdf.SP20120129.R

Notes:

  • This CDF was created by taking the common probes between v14.1 of the Gene (ENSG) and Exon (ENSE) annotations from Brainarray (U Michigan).
  • Probes were placed into 289,999 groups based on common exon mappings.
  • Probes were placed into 35,202 units based on gene annotations.
  • Every probe maps uniquely to one group.
  • Every group maps uniquely to one unit.
  • Some groups contain more than 4 probes. For a visual summary see this.
